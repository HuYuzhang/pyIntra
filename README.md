### 11.17
* 之前得到的预测图像非常奇怪，觉得可能是reshape出了问题，检查了很久，发现是被MATLAB坑到了。MATLAB是
列优先的，所以reshape会出现问题，必须在这里进行reshape的时候指定order='F'保证列优先。
* 另外发现了很多非常没有意义的图片，整张图片几乎都是没有任何变化的。所以之后应该要考虑对于方差小于或者
大于阈值的数据进行过滤（太复杂可能也学不出来？）
